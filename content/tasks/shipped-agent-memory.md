## What I Built

**agent-memory (memctl)** - Memory management CLI for AI agents

### Features
- `memctl list` - List memory files by recency
- `memctl search` - Full-text search across all files
- `memctl summary` - Stats and topic extraction
- `memctl gaps` - Find missing daily entries
- `memctl touch` - Create today's file from template
- `memctl digest` - AI-powered summary via Gemini
